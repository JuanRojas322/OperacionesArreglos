# Operaciones con Arreglos Unidimensionales 

## Objetivo del Proyecto
El objetivo de este proyecto es poner en práctica los conocimientos de la Unidad 1 sobre arreglos unidimensionales en Java. El programa solicita al usuario llenar dos arreglos de 7 posiciones, calcula la diferencia entre ambos almacenando el resultado en un tercer arreglo, y finalmente muestra los datos resultantes junto con su promedio.

## Instrucciones de Ejecución
1. Asegúrate de tener instalado Java (JDK 8 o superior).
2. Clona este repositorio en tu máquina local:
   git clone https://github.com/JuanRojas322/OperacionesArreglos.git
3. Navega hasta la carpeta del proyecto.
4. Compila el archivo Java:
   javac OperacionesArreglos.java
5. Ejecuta el programa:
   java OperacionesArreglos
